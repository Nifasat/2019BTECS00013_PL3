Program to convert Centigrate to Fahrenheit

Code in Python

        
            celsius = float(input("Enter temperature in celsius: "))
            fahrenheit = (celsius * 9/5) + 32
            print('%.2f Celsius is: %0.2f Fahrenheit' %(celsius, fahrenheit))
        
        
            
                
Sample Input:

                Enter temperature in Celsius: 37
                
Sample Output:

                37.00 Celsius is: 98.60 Fahrenheit

Variables used in this program


celsius as the temperature in Celsius. fahrenheit as the temperature in Fahrenheit.

Time Complexity:

We are using the formula to convert Centigrate to Fahrenheit. So the time complexity of the above code is O(1).


We are using constant space, so Space complexity is O(1)
